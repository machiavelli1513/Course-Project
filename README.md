# Course-Project
Project for Advanced Scientific Programming with Python.

**Description:**
---
My project will be about reducing the time needed to manipulate data from a fission code event-by-event file. \\
The event-by-event file has 2 million lines of data and the manipulation consists of performing arithmetic on the data columns. \\

**Method**
---
Possible approaches are multiprocessing or pre-compiling the function.

**Evaluation of result**
---
The result will be evaluated by measuring the time needed to perform the actions.
